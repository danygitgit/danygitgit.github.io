快速安装remarkable

### 前言
这是自己第一次用Remarkable 没有过多的感受和体会，就单纯的记录一下如下在Linux平台安装Remarkable

### 安装过程
****
```
1. 下载安装包 
http://remarkableapp.github.io/linux/download.html
2. 在安装包目录下打开终端
sudo dpkg  -i remarkable_1.62_all.deb
3. 补上依赖
sudo apt-get install -f
4. 运行
remarkable &
```
